================================================================================
🔌 API接口
================================================================================

1. POST /api/v1/materials/{materialId}/classify
   → 分析并分类资料

2. GET /api/v1/materials/{materialId}/keywords
   → 获取资料关键词

3. POST /api/v1/materials/{materialId}/suggest-tags
   → 获取标签建议

4. POST /api/v1/materials/classification-logs/{logId}/accept
   → 接受分类建议

5. POST /api/v1/materials/classification-logs/{logId}/reject
   → 拒绝分类建议
